# Tampermonkey-skin
油猴/篡改猴面板皮肤美化 

# 使用方法
说明Tampermonkey我们称之为篡改猴，以下统称篡改猴<br/>
1、打开篡改猴面板-点击【管理面板】-点击【设置】-在面板中找到配置模式，把配置模式的【新手】改为【高级】<br/>
2、此时下方外观选项多了【用户css】<br/>
3、把此【Tampermonkey-skin】中的代码粘贴进去保存即可<br/>
